README

![Chai Documentation](http://chaijs.com/public/img/chai-logo.png)

Chai is a BDD / TDD assertion library for node and the browser that can be delightfully paired with any javascript testing framework.

For more information or to download plugins, view the documentation.

![Build Status](https://travis-ci.org/chaijs/chai.png?branch=master)

![Selenium Test Status](https://saucelabs.com/browser-matrix/chaijs.svg)

Plugins

Chai offers a robust Plugin architecture for extending Chai's assertions and interfaces.

  • Need a plugin? View the official plugin list.
  • Have a plugin and want it listed? Open a Pull Request at chaijs/chai-docs:plugin.js.
  • Want to build a plugin? Read the plugin api documentation.

Related Projects

  • chaijs / assertion-error: Custom Error constructor thrown upon an assertion failing.
  • chaijs / deep-eql: Improved deep equality testing for Node.js and the browser.

#CSSwhat ![Build Status](https://secure.travis-ci.org/fb55/CSSwhat.png?branch=master)

a CSS selector parser

Example

```js require('CSSwhat')('foo[bar]:baz')

~> [ [ { type: 'tag', name: 'foo' }, { type: 'attribute', name: 'bar', action: 'exists', value: '', ignoreCase: false }, { type: 'pseudo', name: 'baz', data: null } ] ] ```

API

__CSSwhat(selector, options) - Parses str, with the passed options.__

The function returns a two-dimensional array. The first array represents subselects separated by commas (eg. sub1, sub2), the second contains the relevant tokens for that selector. Possible token types are:

tagname div `{ type: 'tag', name: 'div' } <tr><td>universal<td> - <td>*<td>{ type: 'universal' } <tr><td>pseudo<td>name,data<td>:name(data)<td>{ type: 'pseudo', name: 'name', data: 'data' } <tr><td>pseudo<td>name,data<td>:name<td>{ type: 'pseudo', name: 'name', data: null } <tr><td>attribute<td>name,action,value,ignoreCase<td>[attr]<td>{ type: 'attribute', name: 'attr', action: 'exists', value: '', ignoreCase: false } <tr><td>attribute<td>name,action,value,ignoreCase<td>[attr=val]<td>{ type: 'attribute', name: 'attr', action: 'equals', value: 'val', ignoreCase: false } <tr><td>attribute<td>name,action,value,ignoreCase<td>[attr^=val]<td>{ type: 'attribute', name: 'attr', action: 'start', value: 'val', ignoreCase: false } <tr><td>attribute<td>name,action,value,ignoreCase<td>[attr$=val]<td>{ type: 'attribute', name: 'attr', action: 'end', value: 'val', ignoreCase: false }`

0.3.0

  • TouchAction/MultiAction update now works like the W3 specs. See the following sample:

```js TouchAction a1 = (new wd.TouchAction(driver)).press({el: el}).release(); a1.perform(); TouchAction a2 = (new wd.TouchAction()).press({el: el}).release(); TouchAction a3 = (new wd.TouchAction()).tap({el: el, x50, y: 50}).wait({ms: 10000}); MultiAction ma = (new wd.MultiAction(driver)).add(a2, a2); ma.perform(); ```
